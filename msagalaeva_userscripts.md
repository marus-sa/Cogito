# Пользовательские сценарии 

## Сценарий 1 - Регистрация пользователя

1. Пользователь выбирает свою функциональную роль(ученик/репетитор/родитель/наставник/администратор)
2. Пользователь вводит логин, с которым он будет заходить в систему
3. Пользователь вводит пароль, с которым он будет заходить в систему
4. Пользователь вводит адрес электронной почты, который будет использоваться в системе
5. Если выбранный логин уже существует в системе, то пользователю сообщается об этом и предлагается придумать новый логин
6. Если пароль содержит менее 6 символов, система сообщает, что пароль должен быть от 6 до 30 символов и пользователь должен придумать новый пароль
7. Если введённый адрес электронной почты не соответствует формату, то система выводит сообщение об ошибке и просит ввести адрес ещё раз
8. Система запрашивает дополнительную информацию для анкеты: 

У репетитора:
*  Класс
* Предметы преподавания
* Удобное время для занятий
  
У ученика:
* Класс
* Необходимые предметы для изучения 1-2
* Удобное время для занятий


## Сценарий 2 - Коммуникация между участниками
1. Открытие раздела "Чаты"
2. Просмотр списка доступных чатов (по ученикам, группам, проектам)
3. Выбор конкретного чата (например, с репетитором, родителем, координатором)
4. Просмотр истории переписки
5. Написание нового текстового сообщения
6. Прикрепление файлов к сообщению (документы, изображения)
7. Добавление/ удаление новых участников в существующий чат администратором 

## Сценарий 3 - Постановка и отслеживание учебных целей ученика(от лица ученика )
Перенести выбор предмета 

1. Открытие раздела "Мои цели"
2. Нажатие "Новая цель"
3. Формулировка цели 
4. Установка срока выполнения цели
5. Регулярное обновление прогресса по подзадачам(со стороны репетитора)[подробнее в сценарии 4]
6. Просмотр успеваемости 
7. Завершение цели, получение бейджа и мотивационного сообщения


## Сценарий 4 - Анализ успеваемости и прогресса (репетитор)
1. Просмотр цели, поставленной учеником
2. Разбивка цели на подзадачи(необходимые темы + контроль знаний)
3. Добавление комментариев о  результате работы на уроке по конкретной подзадачи
4. Отметка освоенных подзадачи 
5. Автоматическое обновление и просмотр успеваемости


## Сценарий 5 - Отслеживание отработанных часов (репетитор)
1. Открытие раздела «Мои часы»
2. Выбор даты и времени занятия
3. Отметка «занятие проведено»
4. Прикрепление ссылок на видео-записи и скриншоты
5. Добавление комментария к занятию (подробнее в 4 сценарии)
6. Автоматическое добавление часов в итоговое число работы
7. Просмотр общей статистики отработанных часов
8. Формирование отчета по часам за выбранный период



## Сценарий 6 - Прикрепление и проверка домашнего задания (репетитор)
1. Открытие странички курса
2. Нажатие кнопки «Домашнее Задание»
3. Прикрепление ссылок с дз к уроку(опционально прикрепление файлов)
4. Выставление дедлайна сдачи
5. Скчивание и просмотр выполненных заданий
6. Проверка и выставление баллов
7. Добавление результата в анализ успеваемости, если закрылась подзадача(сценарий 4)


## Сценарий 7 - Работа с расписанием занятий
1. Открытие раздела «Календарь занятий»
3. Просмотр расписания на текущую неделю
4. Редактирование расписания 
5. Переключение между режимами просмотра (день / неделя / месяц)
6. Выбор конкретного занятия
7. Просмотр информации о занятии (предмет, время, участники)
8. Прикрепление ссылки(только для репетитора)
9. Переход по ссылке для подключения к онлайн-занятию


## Сценарий 8 - Просмотр прогресса ребенка (родитель)
1. Открытие личного кабинета родителя
2. Переход в раздел «Прогресс ребенка»
3. Просмотр текущих целей ученика
4. Просмотр графиков успеваемости по предметам
5. Просмотр комментариев репетитора по последним занятиям
6. Просмотр назначенных домашних заданий
7. Переход в чат с репетитором для обсуждения результатов



## Сценарий 9 - Контроль занятий наставником(наставник/глава проекта)
1. Переход в раздел «Все занятия проекта»
2. Выбор нужного занятия из списка
3. Просмотр информации о репетиторе и ученике
4. Просмотр прикрепленных записей занятия и материалов
5. Оценка качества проведенного занятия
6. Отправка комментария репетитору при необходимости
7. Добавление отметки о проверке занятия в системе
