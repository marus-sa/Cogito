[Проблемное поле]

У исполнителей-репетиторов: 
Когда новый репетитор приходит в проект, он сталкивается с непониманием, как устроена работа преподавателя в благотворительном проекте. Ему непонятен процесс связи с учениками, отправки видео-записей(репетиторы обязаны прикреплять видео-записи уроков и скрины начала и конца), возникает сложность в поиске инструментов для проведения занятий. В конечном итоге, ему необходимо использовать множество приложений, в которых ученики путаются, а именно: диск с материалами, гугл формы с дз(например, загруженные в классруме), а если репетитор решает не использовать приложения для этого, то появляется куча файлов, в которых тоже возникает путаница.

У исполнителей отдела интеграции:
Множество чатов, куда репетиторы кидают видео-записи и скрины, нет структуризации.

У учеников:
Ученик путается в приложениях или файлах, теряет необходимые материалы, некоторые ученики не обладают достаточной цифровой грамотностью, поэтому процесс "скачай файл с диска, открой, заполни, сохрани, переименуй, загрузи в форму" вызывает стресс и отнимает время от учебы. Также, для поддержания мотивации важно видеть процесс.

У родителей:
У учеников младших классов за процессом обучения следят родители, следовательно, с проблемами учеников, описанными выше, сталкиваются взрослые. 

Многим родителям важно видеть результат, поэтому они постоянно интересуются, как прошло занятие, но иногда они могут бояться нарушить границы, написав/позвонив слишком поздно. 


Маркеры проблемы:
- отсутствие единой экосистемы;
- избыток необходимых приложений для обучения;
- нехватка обратной связи о прошедшем занятии;
- сложность отправки записи занятий;
- сложность понимания процесса взаимодействия с наставником(наставник -  человек, которому отправляем записи занятий; это необходимо для контроля проведения занятий);
- отсутствие необходимого уровня цифровой грамотности

[Образ продукта]

Веб-приложение Cogito - это единая цифровая экосистема для образовательного волонтерства. Где все процессы - от первого знакомства с репетитором до анализа глобальной статистики проекта - происходят в одном месте, логично и удобно.
Оно создано на языке программирования Python с использованием фреймворка Flask для бэкенда и JavaScript с фреймворком Vue.js для фронтенда. 

Приложение работает через веб-сервер Nginx в облачной инфраструктуре. Пользователи (ученики, родители, репетиторы, координаторы) открывают его через браузер как обычный сайт, заходят под своим логином через систему аутентификации Keycloak и работают в соответствии со своей ролью в образовательном процессе.

Проект изначально рассчитан для работы ~ 300 человек

Имеется 5 функциональных ролей: администратор, наставник, репетитор, ученик, родитель

Основная база данных - PostgreSQL, файловое хранилище - S3-совместимое облачное хранилище для видео, скриншотов и учебных материалов. Для разработки и тестирования используется Docker-контейнеризация.

Доступ: Через интернет по защищенному протоколу HTTPS, с возможностью доступа как с компьютеров, так и с мобильных устройств через адаптивный интерфейс.

Интерфейс: веб-приложение (PWA) на Vue.js с адаптивным дизайном. Открывается в любом современном браузере с поддержкой офлайн-режима для основных функций.

Основные функции:

1. Система пользователей с ролями: ученик, родитель, репетитор, наставник, адмін
2. Прикрепление видео-записей и скринов занятий
3. Интегрированная система общения: чаты между всеми участниками процесса
4. Работа с домашними заданиями: назначение, выполнение, проверка, обратная связь
5. Учет волонтерских часов: автоматический подсчет, формирование отчетов
6. Система целей и прогресса: постановка учебных целей, разбивка на подзадачи, отслеживание выполнения
7. Визуальная аналитика: графики успеваемости учеников, статистика занятий, прогресс по темам
8. Мобильная доступность: адаптивный интерфейс, работа через браузер на любом устройстве

Продукт рассчитан на использование в рамках благотворительных образовательных проектов с волонтерами - репетиторами, обеспечивая полный цикл организации образовательного процесса от первого знакомства с репетитором до анализа результатов и формирования отчетности. Cogito позволяет заменить около 4 приложений единой интегрированной платформой, экономя время и повышая эффективность работы всех участников образовательного процесса.
