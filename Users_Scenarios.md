Пользовательские сценарии

1. Регистрация 
1) пользователь открывает приложение
2) пользователь выбирает свою функциональную роль 
3) пользователь вводит свои данные (обязательно почту)
4) пользователь устанавливает пароль
5) пользователь проходит короткое обучение по навигации
6) Система запрашивает дополнительную информацию для анкеты:
у репетитора:
- Класс
- Предметы преподавания
- Удобное время для занятий
у ученика:
- Класс
- Необходимый предмет для изучения 
- Удобное время для занятий

2. Коммуникация между участниками
1) Открытие раздела "Чаты"
2) Просмотр списка доступных чатов (по ученикам, группам, проектам)
3) Выбор конкретного чата (например, с репетитором, родителем, координатором)
4) Просмотр истории переписки
5) Написание нового текстового сообщения
6) Прикрепление файлов к сообщению (документы, изображения)
7) Добавление новых участников в существующий чат

3. Постановка и отслеживание учебных целей
1) Открытие раздела "Мои цели"
2) Нажатие "Новая цель"
3) Формулировка цели (конкретная, измеримая, достижимая)
4) Установка срока выполнения цели
5) Разбивка цели на подзадачи и контрольные точки
6) Регулярное обновление прогресса по подзадачам(со стороны ученика и репетитора)
7) Просмотр прогресса и статистики достижения
8) Завершение цели, получение бейджа и мотивационного сообщения

4. Анализ успеваемости и прогресса (репетитор)
1) Добавление результатов работы на уроке
2) Ввод данных: тип работы, дата, баллы, комментарии
3) Отметка освоенных тем и зон роста
4) Автоматическое обновление графиков успеваемости
5) Просмотр динамики прогресса за выбранный период
6) Родитель просматривает статистику, по вопросам - коммуницимует с преподавателем

5. Отслеживание отработанных часов репетитором
1) Открытие раздела "Мои часы"
2) Выбор даты занятия
3) Отметка - занятие проведено
4) Прикрепление видео-записей и скринов
5) Добавление часов в итоговое число работы
